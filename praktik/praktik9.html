Praktik 9: Remote Desktop & Remote Assistance
Mengakses komputer lain dalam jaringan menggunakan fitur remote desktop.
  1. Aktifkan fitur Remote Desktop pada komputer target.
  2. Catat IP Address komputer target.
  3. Buka Remote Desktop Connection di komputer client.
  4. Masukkan IP Address dan login.
  5. Kendalikan komputer target dari jarak jauh.
