Praktik 7: Instalasi Sistem Operasi Jaringan
Instalasi Linux Server dan konfigurasi dasar layanan jaringan (SSH, FTP, dsb).
  1. Siapkan installer Linux Server (misal Ubuntu Server).
  2. Lakukan instalasi pada komputer/server.
  3. Konfigurasi IP Address dan hostname.
  4. Install layanan jaringan (SSH, FTP, dsb) sesuai kebutuhan.
  5. Uji layanan dari komputer client.
