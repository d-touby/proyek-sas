Praktik 3: Sharing File & Folder di Jaringan LAN
Berbagi file antar komputer dalam satu jaringan lokal menggunakan Windows Sharing.
  1. Pilih file/folder yang ingin dibagikan.
  2. Klik kanan, pilih "Properties" > "Sharing".
  3. Aktifkan sharing dan atur izin akses.
  4. Akses file dari komputer lain melalui jaringan.
