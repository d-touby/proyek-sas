Praktik 4: Konfigurasi Wireless Router & Hotspot
Setting SSID, password, dan keamanan pada perangkat wireless router di kelas.
  1. Hubungkan komputer ke router via kabel LAN.
  2. Buka browser dan akses IP router (misal 192.168.1.1).
  3. Login ke halaman admin router.
  4. Atur SSID, password, dan mode keamanan WiFi.
  5. Simpan pengaturan dan uji koneksi WiFi.
