Praktik 5: Troubleshooting Jaringan LAN
Mendeteksi dan memperbaiki masalah koneksi jaringan menggunakan command prompt dan tools jaringan.
  1. Cek koneksi fisik kabel dan perangkat.
  2. Gunakan perintah ping untuk menguji koneksi.
  3. Periksa pengaturan IP Address dan subnet mask.
  4. Gunakan ipconfig atau ifconfig untuk cek konfigurasi.
  5. Restart perangkat jika perlu.
